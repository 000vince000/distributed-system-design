## Workflows
- POST /auth
  Requirement: User authentication
  Auth Service
    - Validate credentials
    - Generate token
  Database
    - Store session
    - Update last login
- GET /data
  Requirement: Data storage
  Load Balancer
    - Route request
    - Check health
  Application Server
    - Process request
    - Format response
  Cache
    - Check cache
    - Update cache
  Database
    - Query data
    - Return results
